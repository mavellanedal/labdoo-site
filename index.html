En que consiste
el PROYECTO

Este proyecto consiste en crear una infraestructura de red para un colegio.
Esta infraestructura constara de un servidor con dos sistemas operativos: Windows Server 2019 y Ubuntu Server 22.04.3 LTSC.
Esto ayudara a que muchos de los niños de paises necesitados peudan llegar a tener un futuro.

Quienes somos?

Somos un grupo de 4 estudiantes del instituto Centre d'estudis pólitecnics de Barcelona.
Nuestra pasión es la informática y queremos que llegue a todo el mundo.
Por lo que hemos decidido involucrarnos en este proyecto para ello. Colaboraremos con la ONG de Labdoo.
